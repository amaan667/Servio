# Archive Directory

This directory contains archived debug artifacts, temporary scripts, and ad-hoc SQL files that were removed during the non-breaking cleanup.

## Structure

- `sql/` - Ad-hoc SQL scripts and debug queries
- `scripts/` - Temporary JavaScript files and debug scripts  
- `debug/` - Debug routes and test endpoints

## Purpose

These files were moved here to:
1. Remove debug overhead from production builds
2. Clean up the codebase without losing historical context
3. Maintain a record of debugging approaches used during development

## Note

These files are not part of the active codebase and should not be referenced in production code.
